# General Rules (CRITICAL DIRECTIVES)

Please strictly adhere to the following operational guidelines when interacting with this project:

- **File Reading Preference**: Give priority to using the safe reading mode (`rg` / ripgrep) to read and search file content, rather than directly outputting massive file contents.
- **Terminal Execution Constraints**: Under normal circumstances, the use of **PowerShell is strictly prohibited**. Only use standard bash/sh commands. If the environment forces the use of PowerShell and it is unavoidable, you MUST meticulously verify the correctness, syntax, and completeness of the PowerShell commands before execution.
